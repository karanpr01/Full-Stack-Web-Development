# DataBase

- DataBase is a place where the user data get store.

- There are two types of Database ``` SQl ``` and ``` NoSQl ```.

- In ``` SQL ``` databases the data are store in tabular format in ``` Columns ``` and ``` Row ``` format.

- In ``` NoSQL ``` data are store in an Object format.

- For example:-
``` 
{
    Name:"prem",
    age: 21
}
```

- ``` MongoDB ``` is a database in the form of ``` NOSQl ``` format.
